# 69 [sw][tool][windows] mobaxterm font setup

安装的字体在设置界面查找不到，直接改 font2，位置在 `C:\Users\...\AppData\Roaming\MobaXterm\MobaXterm.ini`

```ini
[Font]
Font2=lilex nerd font
FontIsBold2=0
FontHeight2=14
FontCharset2=0
FontQuality2=3
Ligatures=0
Charset2=UTF-8
LockRowsCols=0
LockedCols=80
LockedRows=24
```
