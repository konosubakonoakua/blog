# 112 [sw][windows] change locale to solve chinese app display issue

> [!TIP]
> 1. <kbd>Win</kbd>+<kbd>i</kbd> to open Setting
> 2. `Setting`->`language&region`->`Administrative language setting`->`Region`->`Change system locale`->`Region Setting`
> 3. Select `Chinese (Simplified, China)`
> 4. Check `Beta: Use Unicode UTF-8 for worldwide language support`
